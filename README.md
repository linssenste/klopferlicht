# klopferlicht


### NES Controller 
The project uses two classic Nintento NES Controllers. 
They are wired as is: 
<img width="1024" height="479" alt="image" src="https://github.com/user-attachments/assets/35fbf07c-81bc-45cd-9575-3915284a8adf" />


#### Controller 1
GND (black) - GND
VCC (white) - 5V
Clock (red) - GPIO27 (13)
Latch (orange) - GPIO17 (11)
Data (yellow) - GPIO22 (15)


#### Controller 2
GND (black) - GND
VCC (white) - 5V
Clock (red) - GPIO13 (33)
Latch (orange) - GPIO6 (31)
Data (yellow) - GPIO19 (35)
